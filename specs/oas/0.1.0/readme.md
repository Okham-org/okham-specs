# Open Assertion Spec (OAS)

**Version**: 0.1.0  
**Status**: DRAFT

OAS defines portable, versioned assertion primitives referenced by OLR rulesets.

- Canonical schema: `https://okham.org/schemas/oas/0.1.0/schema.json`
- Intended to be used with:
  - OLR (rulesets): `https://okham.org/schemas/olr/0.1.0/schema.json`
  - OVR (reports): `https://okham.org/schemas/ovr/0.1.0/schema.json`
