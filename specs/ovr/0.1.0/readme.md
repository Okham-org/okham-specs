# Open Validation Report (OVR)

**Version**: 0.1.0  
**Status**: DRAFT  
**Category**: Tooling & Format Standard

---

## Problem

Validation and lint tools produce incompatible outputs, making results hard to compare, aggregate, or automate.

---

## Scope

OVR defines:
- A portable JSON report format for linting and validation
- Standard severity levels and issue metadata
- Run metadata for reproducibility

---

## Maturity

OVR/OLR is **DRAFT** for v0.0.1. Breaking changes are allowed. Schemas may tighten further in minor or major releases.
