## Problem

Reusable snippets (UI patterns, capability fragments, data transforms) lack a common registry format, making discovery and validation inconsistent across tools.

---

## Scope

OSR defines a registry format for:
- Snippet metadata (id, name, version, license)
- Snippet content payloads
- Language and domain tags

---

## Maturity

OSR is **DRAFT** for v0.0.1. Breaking changes are allowed. Schemas may tighten further in minor or major releases.
